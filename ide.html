Types:

string

A string is a combination of characters surrounded by quotes. EX: "hello"

integer

An integer is a whole number. Numbers with decimals are always rounded down. EX: 3

Operators:

+

Adds two variables. You can only add strings and numbers. EX: 2+2

-

Subtracts two variables. You can only subtract numbers. EX: 5-4

Commands:

print

Prints the following variable. You can only print strings and numbers. EX: print "hello"
